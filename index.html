O coeficiente c: onde tudo começa

Antes de qualquer dinâmica, toda situação tem um ponto de partida. Esse ponto é o coeficiente c.

H15 H19 Valor Inicial · f(0)
🧠 Conceito

O que é o coeficiente c?

Em f(x) = ax² + bx + c, o coeficiente c é o termo independente — aquele que não multiplica nenhuma potência de x. Para saber o que ele representa geometricamente, basta substituir x = 0:

f(0) = a·0² + b·0 + c = c Os termos com x somem e sobra apenas c

Isso significa que c é sempre o valor da função quando x = 0. Geometricamente, é o ponto exato onde a parábola cruza o eixo y.

💡 Como memorizar x = 0 representa o instante inicial: dia zero, hora zero, posição de partida. Nesse momento a função vale exatamente c. Por isso, c é sempre o valor inicial da situação modelada.
⚠️ Confusão clássica no ENEM c não é o máximo, nem a média, nem o valor final. É exclusivamente o f(0) — o valor quando x = 0. Se uma questão perguntar "qual era o estoque no início?" e você ver uma expressão do tipo ax² + bx + c, a resposta é diretamente c.
⚡ Situação motivadora

Campanha de vacinação

Uma farmácia registrou que seu estoque E de vacinas (doses) em função dos dias d segue:

E(d) = −3d² + 6d + 120

Qual era o estoque no dia d = 0 (início da campanha)?

  • 1Substituir d = 0 na expressão: E(0) = −3·(0)² + 6·(0) + 120
  • 2Os termos com d viram zero: E(0) = 0 + 0 + 120
  • 3Resta apenas o coeficiente c: E(0) = 120 doses
✓ Conclusão c = 120 é o estoque inicial — o valor antes de qualquer distribuição. Isso não é coincidência: ocorre com qualquer função do 2º grau, independente dos valores de a e b.
🔬 c no gráfico — regra visual

Onde a parábola cruza o eixo y

O valor de c desloca a parábola inteira para cima ou para baixo, sem alterar sua forma nem sua abertura. Apenas o ponto de cruzamento com o eixo y muda.

Sinal de cCruzamento com eixo yExemplo
c > 0Acima da origemf(x) = x² − 4x + 3 → cruza em y = 3
c = 0Na origem (0, 0)g(x) = 2x² − 5x → passa pela origem
c < 0Abaixo da origemh(x) = −x² + 2x − 4 → cruza em y = −4
💡 Leia diretamente Numa questão ENEM com parábola desenhada, para saber o sinal de c basta olhar onde ela cruza o eixo y: acima → c > 0 · na origem → c = 0 · abaixo → c < 0. Sem cálculo nenhum.
⚡ Exemplos Resolvidos

Identificando c em contextos reais

  • af(x) = 2x² − 5x + 7
    c = 7 → f(0) = 7. Se x for o tempo em dias, a situação começa com valor 7.
  • bg(x) = −x² + 4x
    Não há termo independente → c = 0 → g(0) = 0. A parábola passa pela origem.
  • cSaldo F(m) = −2m² + 8m + 5.000
    c = 5.000 → F(0) = R$ 5.000,00. Esse é o depósito inicial antes de qualquer movimentação.
  • dAudiência A(d) = −d² + 6d + 4
    c = 4 → A(0) = 4 centenas. Era a audiência base antes da campanha começar.
💭 Estoque de produto: Q(t) = −0,5t² + 3t + 80. Qual era o estoque no início (t = 0) e o que representa na expressão? (clique para ver)

Q(0) = −0,5·(0)² + 3·(0) + 80 = 80 unidades

Na expressão, 80 é o coeficiente c — o termo independente. Representa o estoque inicial antes de qualquer variação temporal.

Atalho Nunca é necessário calcular se você perceber que os termos com t sempre viram zero quando t = 0. O resultado sempre é c.

Questão 1 de 4Básico

Para cada função abaixo, calcule f(0) e identifique o coeficiente c:

Funçãof(0) = ?c = ?
f(x) = 3x² − 2x + 5??
g(x) = −x² + 4x??
h(x) = 2x² + 7??

Qual das afirmações está correta?

f(0) = 3·(0)² − 2·(0) + 5 = 0 − 0 + 5 = 5 → c = 5
g(0) = −(0)² + 4·(0) = 0 + 0 = 0 → c = 0 (sem termo independente; parábola passa pela origem)
h(0) = 2·(0)² + 7 = 0 + 7 = 7 → c = 7
Gabarito: C — f(0)=5, g(0)=0, h(0)=7
⚠️ Distrator D: g(0)=4 é erro de quem lê o coeficiente b em vez de calcular f(0). Distrator E: h(0)=2 é erro de quem lê o coeficiente a.
Questão 2 de 4Básico
📺 Uma empresa de streaming registrou a audiência A (em centenas de usuários) de um canal ao longo dos dias d de campanha publicitária:
A(d) = −d² + 6d + 4

No dia de lançamento da campanha (d = 0), qual era a audiência base e o que ela representa na expressão?

A(0) = −(0)² + 6·(0) + 4 = 0 + 0 + 4
c = 4 é o termo independente → audiência base antes da campanha = 400 usuários
Gabarito: A
⚠️ Distrator B: confunde c com b. Distrator E: somar coeficientes dá o valor de f(1), não f(0).
Questão 3 de 4Médio
💉 Secretaria de Saúde do Maranhão — campanha de vacinação contra dengue. O estoque E de doses (em centenas) em função dos dias d desde o início da distribuição é modelado por:
E(d) = −3d² + 6d + 120

No dia de início da distribuição (d = 0), qual era o estoque e o que esse valor representa na expressão algébrica?

E(0) = −3·(0)² + 6·(0) + 120 = 0 + 0 + 120
c = 120 → estoque inicial = 12.000 doses (120 centenas)
Gabarito: D
⚠️ Distrator E: E(1) = −3+6+120 = 123, não E(0). Distrator B: b indica inclinação inicial, não o valor inicial.
Questão 4 de 4Médio
🎓 FNDE — Programa de Bolsa Universitária. O saldo F (em reais) de uma bolsa em função dos meses m após o início do semestre é modelado por:
F(m) = −2m² + 8m + 5.000

No momento do depósito inicial (m = 0, antes de qualquer gasto), qual era o saldo disponível e o que esse valor representa na expressão?

F(0) = −2·(0)² + 8·(0) + 5.000 = 0 + 0 + 5.000
c = 5.000 → saldo inicial = R$ 5.000,00
Gabarito: A

Ler o gráfico e descobrir os coeficientes

Sem fórmula — apenas olhando a parábola. Três passos, três coeficientes.

H20 H21 Leitura de Gráfico
🧠 Os 3 Passos

Como ler a, b e c diretamente do gráfico

Cada coeficiente deixa uma "impressão digital" visual na parábola. Você já sabe ler a (Aula 2) e c (Aula 3). Agora juntamos os três — e o novato é o coeficiente b.

1
Sinal de a
Olhe a concavidade:
∩ (cúpula) → a < 0
∪ (tigela) → a > 0
2
Sinal de b
Use xv = −b/2a → b = −2a·xv
a e xv com mesmo sinal → b < 0
a e xv com sinais opostos → b > 0
xv = 0 → b = 0
3
Sinal de c
Veja onde cruza o eixo y:
Acima da origem → c > 0
Na origem (0,0) → c = 0
Abaixo da origem → c < 0
⚠️ O caso especial: xv = 0 Quando o vértice está sobre o eixo y, b = −2a·0 = 0. A parábola é simétrica em relação ao eixo y, e o termo bx desaparece da expressão.
💡 Por que b = −2a·xv funciona? Da fórmula do vértice: xv = −b/(2a). Isolando b: b = −2a·xv. O produto de dois números tem sinal negativo quando os fatores têm sinais iguais, e positivo quando têm sinais opostos — daí a regra do Passo 2.
⚡ Exemplos Resolvidos — Aplicando os 3 Passos

Dois casos completos

  • Ex1 Parábola ∩, vértice em xv < 0, cruza eixo y acima da origem:
    Passo 1: ∩ → a < 0
    Passo 2: xv < 0 e a < 0 → mesmo sinal → b = −2·(neg)·(neg) = negativo → b < 0
    Passo 3: cruza acima → c > 0
  • Ex2 Parábola ∩, vértice em xv > 0, cruza eixo y abaixo da origem:
    Passo 1: ∩ → a < 0
    Passo 2: xv > 0 e a < 0 → sinais opostos → b = −2·(neg)·(pos) = positivo → b > 0
    Passo 3: cruza abaixo → c < 0
  • Ex3 Parábola ∪, vértice em xv = 0, cruza eixo y acima da origem:
    Passo 1: ∪ → a > 0
    Passo 2: xv = 0 → b = −2a·0 = b = 0
    Passo 3: cruza acima → c > 0
💭 Como memorizar a regra do Passo 2 de forma definitiva? (clique)

b = −2a·xv. O sinal de b é sempre oposto ao produto a·xv. Veja:

axva·xvb = −2·(a·xv)
− −+negativo
−+−positivo
+−−positivo
+++negativo
Resumo b tem o mesmo sinal que o produto a·xv negado. Ou simplesmente: se a e xv têm sinais iguais → b é negativo. Se têm sinais opostos → b é positivo.
💭 Parábola ∪ com vértice em xv > 0 e c < 0. Quais os sinais? (clique)

Passo 1: ∪ → a > 0

Passo 2: xv > 0 e a > 0 → mesmo sinal → b = −2·(pos)·(pos) → b < 0

Passo 3: c < 0 (dado) → c < 0

Resultado: a > 0, b < 0, c < 0

Questão 2 de 4Básico

Uma parábola tem concavidade para baixo, vértice à esquerda do eixo y (xv < 0) e cruza o eixo y acima da origem. Os sinais de a, b e c são:

Passo 1: ∩ → a < 0 ✓
Passo 2: xv < 0 e a < 0 → mesmo sinal → b = −2·(neg)·(neg) = negativo → b < 0 ✓
Passo 3: cruza acima → c > 0 ✓
Gabarito: B — a < 0, b < 0, c > 0

Questão 3 de 4Médio
🌡️ INMET — Temperatura em Belém do Pará. A temperatura T (°C) ao longo do dia é modelada por uma parábola com as seguintes características:
• Concavidade para baixo (temperatura sobe, atinge máximo e cai);
• Vértice após o meio-dia, portanto xv > 0 (onde x = hora do dia);
• Temperatura às 0h é positiva, portanto c > 0.

Com base nessas informações, os sinais de a, b e c de T(h) são:

Passo 1: temperatura sobe, atinge máximo e cai → parábola ∩ → a < 0 ✓
Passo 2: xv > 0 e a < 0 → sinais opostos → b = −2·(neg)·(pos) = positivo → b > 0 ✓
Passo 3: T às 0h é positiva (c > 0) — dado explícito ✓
Gabarito: B — a < 0, b > 0, c > 0
⚠️ Distrator C: aplica "mesmo sinal" quando deveria ser "sinais opostos" no Passo 2.
Questão 4 de 4Difícil — Padrão ENEM
⚡ ANEEL — Relatório de consumo de energia elétrica de uma indústria ao longo do mês. O gráfico do consumo E (kWh) em função do dia d do mês é uma parábola com as seguintes características observadas:
• Concavidade para baixo — o consumo sobe, atinge pico e diminui;
• Vértice localizado à direita do eixo y: xv > 0;
• A parábola cruza o eixo y abaixo da origem: c < 0.

Com base no gráfico descrito, assinale a alternativa com os sinais corretos de a, b e c de E(d) = ad² + bd + c:

Passo 1: consumo sobe, atinge pico e cai → ∩ → a < 0 ✓
Passo 2: xv > 0 e a < 0 → sinais opostos → b = −2·(neg)·(pos) = positivo → b > 0 ✓
Passo 3: cruza abaixo da origem → c < 0 ✓ (dado explícito)
Gabarito: B — a < 0, b > 0, c < 0
⚠️ Distrator E: erro duplo — acerta a e c mas erra b (confunde a regra de sinais do Passo 2). Esse é o distrator mais perigoso.
