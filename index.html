Princípio Fundamental da Contagem

Antes de contar tudo, aprenda a multiplicar escolhas. É mais simples do que parece.

H1H2H3
🧠 Conceito

O Princípio Multiplicativo

Se uma tarefa pode ser realizada em k etapas independentes, sendo n₁ as formas de realizar a 1ª etapa, n₂ a 2ª, ..., o número total de maneiras é:

Total = n₁ × n₂ × n₃ × … × nₖ Cada etapa é independente das demais
💡 A intuiçãoPara cada escolha da primeira etapa, você tem todas as opções da segunda etapa. Para cada combinação dessas duas, você tem todas as da terceira. Por isso se multiplica — não se soma.
⚠️ Quando NÃO usar o PFCO PFC assume que cada etapa é independente e que as opções não se excluem. Se uma escolha elimina opções em outra etapa (ex: não pode repetir letras, não pode usar a mesma pessoa em dois cargos), é preciso atenção redobrada na contagem de cada etapa.

Fatorial e Permutação Simples

Quando a ordem importa e usamos todos os elementos, entra em cena a permutação.

🧠 Fatorial — o produto que cresce rápido

O que é n! ?

O fatorial de n (escrito n!) é o produto de todos os inteiros positivos de 1 até n:

n! = n × (n−1) × (n−2) × … × 2 × 1 Por definição: 0! = 1
n1234567
n!126241207205.040
⚠️ Fatorial cresce absurdamente rápido!10! = 3.628.800 e 20! ≈ 2,4 × 10¹⁸. O ENEM nunca pede o cálculo de fatoriais grandes — sempre há simplificação antes de resolver.

🧠 Permutação Simples

Quando ordem importa e usa-se tudo

A permutação simples conta de quantas maneiras distintas podemos ordenar n elementos diferentes, usando todos eles:

Pₙ = n! n = total de elementos · todos são usados · ordem importa
💡 Por que é igual a n! ?Para o 1º lugar: n escolhas. Para o 2º: n−1. Para o 3º: n−2. ... Para o último: 1 escolha. Pelo PFC: n × (n−1) × … × 1 = n!
⚠️ Ordem importa ou não?AB e BA são a MESMA dupla de pessoas (combinação) mas ESCALAÇÕES diferentes (permutação). A pergunta-chave: trocar a posição de dois elementos gera um resultado diferente? Se sim, use permutação ou arranjo.

Arranjo

Ordem importa, mas não usamos todos — escolhemos uma parte e a ordenamos.

🧠 Conceito

Arranjo Simples de n elementos, p a p

O arranjo conta de quantas formas podemos escolher e ordenar p elementos de um conjunto de n elementos distintos (p ≤ n). A ordem dos escolhidos importa.

A(n,p) = n! / (n−p)! n = total de elementos · p = quantos são escolhidos · ordem importa
💡 Leitura intuitivaA(n,p) = n × (n−1) × (n−2) × … × (n−p+1). São exatamente p fatores, começando em n e descendo. Ex: A(6,3) = 6×5×4 = 120.
⚠️ Arranjo vs PermutaçãoPermutação é um caso especial de arranjo onde p = n, ou seja, A(n,n) = n!/0! = n!. Se usar todos os elementos, é permutação. Se usar só parte, é arranjo.

Combinação

Quando a ordem NÃO importa — só o grupo escolhido.

🧠 Conceito

Combinação Simples

A combinação conta de quantas formas podemos escolher p elementos de n disponíveis, sem se importar com a ordem dos escolhidos. Trocar a posição não cria um novo grupo.

C(n,p) = n! / (p! × (n−p)!) também escrita como Cₙᵖ ou "n escolhe p"
💡 De onde vem a fórmula?C(n,p) = A(n,p) / p! — pegamos o arranjo (onde ordem importa) e dividimos por p! (o número de ordens do mesmo grupo que não queremos contar duas vezes).
🧠 O Grande Comparativo

PFC, Permutação, Arranjo ou Combinação?

Permutação Pₙ = n!

Usa todos os n elementos. Ordem importa.

Ex: anagramas, filas

Arranjo A(n,p)

Usa p de n elementos. Ordem importa.

Ex: pódio, senha, chapas

Combinação C(n,p)

Usa p de n elementos. Ordem NÃO importa.

Ex: comissões, grupos, times

⚠️ A pergunta-chave para decidir"Se eu trocar dois elementos de posição, cria uma situação nova?" Se SIM → Arranjo/Permutação. Se NÃO → Combinação.

⚡ Exemplos Resolvidos
  • Ex1Comissão de 3 em turma de 10: C(10,3) = 10!/(3!×7!) = (10×9×8)/(3×2×1) = 120
  • Ex2Mão de 5 cartas em baralho de 52: C(52,5) = 2.598.960
  • Ex3Time de 11 jogadores de 18 convocados: C(18,11) = C(18,7) = 31.824
💭 Por que C(18,11) = C(18,7)? Qual a propriedade por trás? (clique)

Propriedade da simetria: C(n,p) = C(n, n−p)

Escolher 11 de 18 é o mesmo que escolher os 7 que ficam de fora. A fórmula confirma: C(18,11) = 18!/(11!×7!) = C(18,7).

Use isso no ENEM! Quando p é grande, calcule C(n, n−p) com p menor — muito mais rápido.
💭 Lanchonete com 8 sabores. Cliente escolhe 2 sabores distintos para o sorvete de casquinha. Quantas duplas? (clique)

Ordem não importa (baunilha+morango = morango+baunilha) → Combinação

C(8,2) = (8×7)/(2×1) = 28 duplas

✏️ Exercícios — Combinação & Revisão Geral

C(n,p) = n! / (p!·(n−p)!) — e o desafio final de distinguir os 4 conceitos.

Questão 1 de 4Básico

Uma turma tem 8 alunos. O professor quer escolher uma comissão de 3 alunos para representar a turma. Quantas comissões distintas podem ser formadas?

Comissão: ordem não importa → Combinação
C(8,3) = 8!/(3!×5!) = (8×7×6)/(3×2×1) = 336/6 = 56
Resposta: C
Questão 2 de 4Básico
🃏 Em um baralho padrão de 52 cartas, quantas mãos de 2 cartas diferentes são possíveis?

Uma mão de cartas não tem ordem — {7♠, K♥} é a mesma mão que {K♥, 7♠}.

Ordem não importa → C(52,2) = (52×51)/(2×1) = 2.652/2 = 1.326 mãos
Resposta: D
Questão 3 de 4Médio — Identifique o conceito

Para cada situação abaixo, identifique o conceito correto:

SituaçãoConceito
Anagramas da palavra BRASIL?
Pódio (1º, 2º, 3º) em corrida com 8 atletas?
Grupo de 4 amigos escolhido de 10?
Número de pratos num menu com 3 entradas e 4 pratos?

Qual sequência de conceitos está correta?

BRASIL (6 letras distintas, usa todas, ordem importa) → Permutação P₆ = 720
Pódio (usa 3 de 8, posição diferente = resultado diferente) → Arranjo A(8,3) = 336
Grupo de amigos (4 de 10, trocar posição não muda o grupo) → Combinação C(10,4) = 210
Pratos (etapas independentes, multiplica) → PFC 3×4 = 12
Resposta: C
Questão 4 de 4Difícil — Padrão ENEM
⚽ (ENEM adaptada) Um técnico de futebol tem um elenco de 20 jogadores: 3 goleiros, 8 defensores, 5 meias e 4 atacantes. Ele precisa escalar um time titular com 1 goleiro, 4 defensores, 4 meias e 2 atacantes. Quantos times titulares distintos podem ser escalados?

Dentro de cada posição, a ordem dos jogadores não importa — apenas quem joga, não onde.

Goleiro: C(3,1) = 3
Defensores: C(8,4) = 8!/(4!×4!) = (8×7×6×5)/(4×3×2×1) = 70
Meias: C(5,4) = 5!/(4!×1!) = 5
Atacantes: C(4,2) = 4!/(2!×2!) = 6
Total (PFC): 3 × 70 × 5 × 6 = 6.300 times distintos
Resposta: D
